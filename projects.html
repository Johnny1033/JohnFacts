This website was inspired by Faye Ellis from A Cloud Guru and her Serverless Calorie-Free Fortune Cookie project. I wanted to make a fun project that would be entirely serverless but also give some fun information about me. Some of the JohnFacts may not be the best, but the state of large language models isn't focused on creating the most humorous of AI. All of these facts were transformed from my resume using the ChatGPT API, with some creative prompt engineering, in a separate jupyter notebook I was running in SageMaker.
